### JWT validation
- JWT (JSON web tokens)는 오늘날 많은 웹 앱에서 인증 컴포넌트로 사용되고 있습니다. JWT는 사용자와 그들의 액세스를 식별하는데 중요한 역할을 합니다.
- JWT validation 기능은 JWT replay attack/ JWT tarmpering 등을 방지할 수 있습니다.
